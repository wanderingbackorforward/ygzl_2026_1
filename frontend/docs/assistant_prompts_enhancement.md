# 对话助手右侧边栏提示词扩展方案

## 改进概述

对对话助手的右侧边栏进行了全面扩展，使提示词更丰富、更智能，并能区分 Chat 和 Agent 两种模式。

## 主要改进

### 1. 类型系统扩展

**文件**: `frontend/src/components/assistant/types.ts`

为 `QuickCommand` 接口添加了新字段：
- `mode?: AssistantMode | 'both'` - 指定提示词适用的模式（chat/agent/both）
- `category?: string` - 分类标签，用于组织和筛选
- `description?: string` - 详细描述，帮助用户理解提示词用途
- `tags?: string[]` - 搜索标签，提升搜索体验

### 2. 提示词配置大幅扩展

**文件**: `frontend/src/components/assistant/config.ts`

#### 原有提示词数量
- 约 20 个基础提示词

#### 扩展后提示词数量
- **60+ 个提示词**，覆盖所有模块和场景

#### 新增提示词类型

**Chat 模式专用提示词**（知识问答，不查询实时数据）：
- 解释沉降原理
- 沉降控制标准查询
- 温度对结构的影响
- 裂缝修复方案咨询
- InSAR 技术原理
- 专业知识问答
- 系统使用帮助

**Agent 模式专用提示词**（自动查询数据并分析）：
- 智能沉降数据分析
- 多点位对比分析
- 自动生成导出报告
- 生成温度热力图
- 裂缝发展速率分析
- 振动频谱分析
- 智能异常诊断
- 多点位趋势预测
- 施工事件影响分析
- 空间关联分析
- 综合风险评估
- 盾构对沉降的影响
- 自定义数据分析

**Both 模式提示词**（两种模式都适用）：
- 生成各类监测报告
- 检查异常点位
- 趋势预测
- 巡检清单
- 工单管理

### 3. UI 组件全面升级

**文件**: `frontend/src/components/assistant/QuickCommandPanel.tsx`

#### 新增功能

**搜索功能**：
- 实时搜索提示词
- 支持标题、描述、标签多维度匹配
- 搜索结果高亮显示

**分类筛选**：
- 动态提取所有分类
- 点击分类快速筛选
- "全部"选项显示所有提示词

**模式感知**：
- 自动根据当前模式（Chat/Agent）筛选提示词
- 顶部显示当前模式说明
- 提示词卡片显示模式标签

**分组显示**：
- 选择"全部"时按分类分组显示
- 每个分类独立展示，便于浏览
- 选择特定分类时平铺显示

**增强的提示词卡片**：
- 显示图标、标题、模式标签
- 显示详细描述
- 显示搜索标签
- 模式标签颜色区分（Agent=紫色，Chat=青色）

**模式说明面板**：
- 底部固定显示模式说明
- Chat 模式：基于知识库回答问题，不查询实时数据
- Agent 模式：自动调用工具查询真实数据并生成分析

### 4. 集成到主面板

**文件**: `frontend/src/components/assistant/AssistantPanel.tsx`

- 将 `currentMode` 参数传递给 `QuickCommandPanel`
- 实现模式切换时自动更新提示词列表

## 使用体验

### Chat 模式
用户切换到 Chat 模式时，右侧边栏会显示：
- 知识问答类提示词
- 标准规范查询
- 原理解释
- 案例学习
- 使用帮助

### Agent 模式
用户切换到 Agent 模式时，右侧边栏会显示：
- 智能数据分析
- 自动查询和可视化
- 多维度对比分析
- 综合诊断和预测
- 自动生成报告

### 搜索和筛选
- 输入关键词快速找到相关提示词
- 点击分类标签查看特定类型的提示词
- 所有筛选实时响应，无需刷新

## 技术亮点

1. **类型安全**：完整的 TypeScript 类型定义
2. **性能优化**：使用 `useMemo` 缓存筛选结果
3. **响应式设计**：自适应不同屏幕尺寸
4. **用户体验**：实时搜索、分类筛选、模式感知
5. **可扩展性**：易于添加新的提示词和分类

## 未来扩展方向

### 1. AI 生成个性化提示词
可以添加一个功能，让 AI 根据用户的历史对话和当前页面上下文，动态生成个性化的提示词建议。

**实现思路**：
```typescript
// 在 QuickCommandPanel 中添加
const [aiSuggestions, setAiSuggestions] = useState<QuickCommand[]>([])

async function generateAISuggestions() {
  const response = await assistantApi.generatePromptSuggestions({
    currentModule,
    currentRole,
    currentMode,
    pageContext,
    recentMessages
  })
  setAiSuggestions(response.suggestions)
}
```

### 2. 用户自定义提示词
允许用户保存自己常用的提示词，并在右侧边栏中显示。

**实现思路**：
- 添加"收藏"按钮到提示词卡片
- 添加"自定义"分类
- 将用户收藏的提示词保存到后端

### 3. 提示词使用统计
记录每个提示词的使用频率，优先显示常用提示词。

**实现思路**：
- 在点击提示词时记录使用次数
- 按使用频率排序显示
- 添加"最常用"分类

### 4. 智能推荐
根据当前页面数据特征，智能推荐最相关的提示词。

**实现思路**：
- 分析当前页面的数据状态（是否有异常、趋势等）
- 根据数据特征推荐相应的分析提示词
- 在顶部显示"智能推荐"区域

## 总结

本次改进大幅提升了对话助手右侧边栏的实用性和智能化程度：
- 提示词数量从 20 个增加到 60+ 个
- 新增 Chat/Agent 模式区分
- 新增搜索和分类筛选功能
- 新增详细描述和标签系统
- 改进 UI 布局和交互体验

用户现在可以更快速地找到合适的提示词，并且能够清楚地区分哪些是知识问答（Chat），哪些是智能分析（Agent）。

# 高级分析模块升级计划 - 完整版

## 项目概述

**项目名称**: 数字孪生地形沉降监测系统 - 高级分析模块升级

**升级目标**: 将"高级分析"模块从**数据展示工具**升级为**智能诊断中心**

**核心价值**:
- 自动告诉用户"哪里有问题"
- 解释"为什么有问题"
- 建议"该怎么办"
- 预测"未来会怎样"

---

## 现状分析

### 已完成（后端ML能力）

✅ **第一阶段ML算法**已全部实现：
- 异常检测（孤立森林）
- 时间序列预测（ARIMA/SARIMA）
- Prophet预测（可选）
- 空间关联分析
- 因果推断（DID/SCM）
- 自动模型选择器

✅ **10个ML API端点**已集成到后端

✅ **测试验证**：5个测试，4个通过

### 存在问题

❌ **前端未调用ML API**：高级分析页面只有图表展示，没有智能分析

❌ **ML功能闲置**：强大的ML能力用户完全看不到

❌ **缺少决策支持**：用户不知道哪里有问题、该怎么办

❌ **数据孤立**：沉降、温度、裂缝、施工事件分开看，没有关联分析

## 升级方案

### 4大智能板块

#### 1. 智能异常诊断 (P0 - 最高优先级)
**用户价值**: 一眼看到所有问题点
- 自动识别所有异常点
- 按严重程度分级（严重/高/中/低）
- 显示异常类型（突变/加速/波动/趋势）
- 异常点地图标注

#### 2. 处置建议 (P0 - 最高优先级)
**用户价值**: 知道该怎么办
- 根据异常自动生成建议
- 按优先级排序（紧急/高/中/低）
- 具体行动方案（巡检/维修/监测/上报）
- 一键生成工单

#### 3. 趋势预测与预警 (P1 - 高优先级)
**用户价值**: 提前发现风险
- 预测未来7/15/30天沉降趋势
- 识别高风险点位
- 预警：哪些点可能超限
- 模型对比（ARIMA vs SARIMA vs Prophet）

#### 4. 关联分析 (P2 - 中优先级)
**用户价值**: 找到根本原因
- 施工事件影响量化
- 温度-沉降关联性
- 空间关联（相邻点位联动）
- 多因素相关性矩阵

## 技术架构

### 前端技术栈

**核心框架**:
- React + TypeScript
- Ant Design / Material-UI（UI组件库）
- ECharts（图表）
- React Query / SWR（数据获取和缓存）

**数据流**:
```
用户打开页面
  -> React Query 自动调用 API
  -> 缓存数据（避免重复请求）
  -> 渲染组件
  -> 用户交互（筛选、排序、点击）
  -> 更新UI
```

// 异常数据
interface Anomaly {
  point_id: string;
  point_name: string;
  date: string;
  settlement: number;
  anomaly_score: number;
  severity: 'critical' | 'high' | 'medium' | 'low';
  anomaly_type: 'spike' | 'acceleration' | 'fluctuation' | 'trend' | 'unknown';

interface Recommendation {
  id: string;

  action: string;
  reason: string;
  related_anomalies: string[];

interface Prediction {
  point_id: string;
  dates: string[];
  forecast: number[];
  lower_bound: number[];
  upper_bound: number[];

  metrics: {
    mae: number;

interface SpatialCorrelation {

    id: string;
    name: string;
    x: number;
    y: number;
  }>;
  correlation_matrix: number[][];
  clusters: number[][];
}

### 错误处理模板

```typescript
const AnomalyDashboard: React.FC = () => {
  const { data, isLoading, error } = useQuery('anomalies', fetchAllAnomalies);

  if (isLoading) {
    return <Spin tip="正在加载异常数据..." />;
  }

  if (error) {
    return (
      <Alert
        message="加载失败"
        description="无法获取异常数据，请稍后重试"
        type="error"
        showIcon
      />
    );
  }

  if (!data || data.length === 0) {
    return <Empty description="暂无异常数据" />;
  }

  return <div>{/* 正常渲染 */}</div>;
};
```

## 性能优化

### 数据缓存

使用 React Query 自动缓存：
```typescript
const { data } = useQuery(
  'anomalies',
  fetchAllAnomalies,
  {
    staleTime: 5 * 60 * 1000, // 5分钟内不重新请求
    cacheTime: 10 * 60 * 1000, // 缓存10分钟
    refetchOnWindowFocus: false // 窗口聚焦时不重新请求
  }
);
```

### 虚拟滚动

大量数据时使用虚拟滚动：

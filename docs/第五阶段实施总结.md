## 技术实现

### 统一模式
所有页面都遵循相同的实现模式：

1. **导入 Hook**
```typescript
import { cachePageData } from '../hooks/usePageContext';
```

2. **添加 useEffect**
```typescript
useEffect(() => {
  if (数据已加载) {
    const dataSnapshot = {
      summary: { /* 关键指标 */ },
      statistics: { /* 统计信息 */ },
      selectedItems: [ /* 选中项 */ ],
    };
    cachePageData('页面标识', dataSnapshot);
  }
}, [依赖项]);
```

3. **页面标识**
- `temperature` - 温度页面
- `cracks` - 裂缝页面
- `vibration` - 振动页面
- `overview` - 总览页面

### 数据存储
- 使用 `localStorage` 存储
- Key 格式: `page_data_<页面标识>`
- 自动序列化为 JSON

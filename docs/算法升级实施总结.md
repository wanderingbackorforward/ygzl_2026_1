## 二、功能详解

### 2.1 智能异常检测

**技术**: 孤立森林（Isolation Forest）

**特点**:
- 无监督学习，无需标注数据
- 8维特征工程（沉降值、日速率、移动平均、标准差、加速度、偏离度等）
- 自动分类异常类型（突变、加速、波动、趋势）
- 严重程度评估（critical/high/medium/low）

**测试结果**:
- 数据点: 52个
- 检测到异常: 3个（5.77%）
- 异常类型: spike（突变）、unknown

**API端点**:
```
GET  /api/ml/anomalies/<point_id>
POST /api/ml/anomalies/batch
```

---

### 2.2 时间序列预测

**技术**: ARIMA + SARIMA

**特点**:
- 自动参数选择（网格搜索最优p, d, q）
- 平稳性检测（ADF检验）
- 季节性建模（SARIMA）
- 95%置信区间

**测试结果**:
- 模型参数: 自动选择
- AIC评分: 模型质量评估
- 预测未来30天

**API端点**:
```
GET /api/ml/predict/<point_id>?model=arima&steps=30
GET /api/ml/predict/<point_id>?model=sarima&steps=30
```

---

### 2.3 自动模型选择

**技术**: 多模型评估 + 自动选择

**支持的模型**:
1. 线性回归（基准模型）
2. ARIMA
3. SARIMA
4. Prophet（可选）

**选择策略**:
- 分析数据特征（趋势强度、波动性、季节性、平稳性）
- 评估所有模型性能（MAE、RMSE、MAPE）
- 自动选择最优模型

**测试结果**:
- 选择的模型: linear_regression
- 评估分数(MAE): 0.418
- 数据特征: 趋势强度=0.810, 波动性=-0.479

**模型对比**:
| 模型 | MAE | RMSE |
|------|-----|------|
| linear_regression | 0.418 | 0.534 |
| arima | 0.768 | 0.952 |
| sarima | 0.522 | 0.602 |

**API端点**:
```
GET /api/ml/auto-predict/<point_id>?steps=30&metric=mae
GET /api/ml/compare-models/<point_id>
```

---

### 2.4 空间关联分析

**技术**: 图论 + 相关性分析

**功能**:
- 构建空间邻接矩阵（基于距离）
- 计算沉降相关性矩阵（皮尔逊相关系数）
- 识别空间聚类（相关性高且距离近的点群）
- 分析异常传播路径

**测试结果**:
- 监测点数量: 25个
- 发现聚类数: 1个
- 聚类1: 包含9个点

**API端点**:
```
GET /api/ml/spatial/correlation?distance_threshold=50
GET /api/ml/spatial/influence/<source_point_idx>
```

---

### 2.5 因果推断（施工事件影响量化）

**技术**: 双重差分法（DID） + 合成控制法（SCM）

**功能**:
- 量化施工事件对沉降的真实影响
- 排除自然趋势的干扰
- 构建反事实预测（如果没有事件会怎样）
- 提供95%置信区间

**应用场景**:
- 爆破施工影响评估
- 开挖作业影响分析
- 重型设备进场影响量化

**API端点**:
```
POST /api/ml/causal/event-impact
Body: {
  "point_id": "S1",
  "event_date": "2024-06-15",
  "method": "DID",
  "window_days": 30
}
```

## 四、API文档

### 4.1 智能预测API

#### 自动选择最优模型预测
```http
GET /api/ml/auto-predict/<point_id>?steps=30&metric=mae
```

**参数**:
- `point_id`: 监测点ID
- `steps`: 预测步数（默认30天）
- `metric`: 评估指标（mae/rmse/mape）

**响应**:
```json
{
  "success": true,
  "point_id": "S1",
  "selected_model": "arima",
  "model_selection_info": {
    "best_score": 0.32,
    "data_characteristics": {
      "data_size": 52,
      "trend_strength": 0.810,
      "volatility": 0.479
    }
  },
  "forecast": {
    "dates": ["2024-01-01", "2024-01-02", ...],
    "values": [5.2, 5.3, ...],
    "lower_bound": [4.8, 4.9, ...],
    "upper_bound": [5.6, 5.7, ...]
  }
}
```

#### 指定模型预测
```http
GET /api/ml/predict/<point_id>?model=arima&steps=30
```

- `method`: 检测方法（isolation_forest/lof）

}

{

## 技术创新点

### 1. 物理约束融合
- PINN模型首次将土体力学方程融入深度学习
- 提升预测的物理可解释性
- 支持多种物理约束（固结理论、速率约束、单调性）

### 2. 多模型集成
- Ensemble整合传统模型和深度学习模型
- 支持Stacking、加权平均、简单平均
- 自动计算模型权重和置信区间

### 3. 可解释性增强
- SHAP分析量化特征贡献
- 支持tree、linear、deep、kernel多种模型
- 提供特征重要性排序和单个预测解释

### 4. 知识图谱构建
- 基于Neo4j的图数据库
- 6种实体类型、8种关系类型
- 支持复杂的图查询和推理

## 遇到的挑战

### 1. Windows编码问题
**问题**: Python中文字符串在Windows环境下编码错误
**解决**: 使用拼音+标记替代中文，或从外部文件读取

### 2. 数据库连接方式
**问题**: 项目使用Supabase HTTP API，不是MySQL
**解决**: 统一使用Supabase REST API方式

### 3. 前端移动端兼容
**问题**: 需要同时支持Web和Android APK
**解决**: 使用API_BASE和STATIC_BASE前缀，避免硬编码路径

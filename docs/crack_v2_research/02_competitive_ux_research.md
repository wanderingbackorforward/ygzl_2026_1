# 竞品UX研究汇总

## 1. SHM行业最佳实践
- **3层信息架构**：总览 → 分区 → 单点详情
- **F/Z扫描模式**：最关键数字放左上角
- **交通灯色彩系统**：绿=安全 黄=警告 红=危险
- **Drawer模式**：详情用右侧抽屉，不用模态框
- **可配置仪表盘**：预设布局（现场巡检/工程师/经理）
- **2024-2025趋势**：AI异常徽章、自然语言摘要、数字孪生集成

## 2. 裂缝监测数字孪生
- **2D平面图为主导航**：点击监测点pin → 查看数据
- **热力图叠加**：裂缝严重度映射为颜色梯度
- **时序趋势图**：宽度(mm)随时间变化 + 阈值线
- **照片叠加+测量**：实际裂缝照片标注宽度/长度
- **创新**：UAV航拍集成、概率寿命预测、虚实同步

## 3. Apple Health — 复杂数据简化呈现
- **3层渐进式披露**：大数字 → 简图 → 全详情
- **收藏/置顶系统**：用户自选4-8个关注指标
- **一致色彩编码**：每类数据固定颜色，全局一致
- **D/W/M/Y时间切换**：每张图都有
- **AI洞察卡片**：自然语言注释放在图表上方
- **极简图表**：少网格线、少标签、大留白

## 4. Linear — 极简主义项目面板
- **内容即界面**：UI chrome几乎不可见
- **键盘优先**：Cmd+K 命令面板
- **列表+详情模式**：左侧密集列表 + 右侧详情面板
- **微型状态指示器**：小圆点而非大徽章
- **8px间距系统**：高信息密度但不拥挤
- **克制的色彩**：灰色为主，颜色出现时立即吸引注意

## 5. Tesla FSD — 实时传感器数据可视化
- **语义色彩编码**：红=即时危险 蓝=规划中 灰=被动/检测到
- **展示系统感知力**：显示所有被检测到的对象 → 建立信任
- **预测轨迹可视化**：蓝色渐变线显示规划路径
- **比例渲染**：物体大小与实际尺寸成比例
- **动画传达状态**：移动的行人动画 vs 静止的行人
- **适应性UI**：根据环境条件调整对比度和动画

## 关键启示（适用于裂缝监测）
1. **先回答问题再展示数据** — "这个裂缝安全吗？" 大字YES/NO
2. **渐进式披露3层** — 总览→列表→详情抽屉
3. **预测性可视化** — 虚线显示"按当前速率X天后达到阈值"
4. **AI洞察卡片** — "DB-003本周增长率提高40%"
5. **动画传达活跃度** — 活跃增长的裂缝呼吸式脉冲
6. **键盘优先** — Cmd+K搜索任意监测点

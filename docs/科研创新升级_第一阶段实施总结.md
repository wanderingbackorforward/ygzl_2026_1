## 二、技术实现详情

### 1. PINN 物理信息神经网络

**文件**: `backend/modules/ml_models/pinn_predictor.py`

**核心创新**:
- 融合Terzaghi固结理论等土体力学方程
- 数据驱动 + 物理约束提升预测精度
- 支持沉降速率约束和单调性约束

**物理定律模块**:
```python
class PhysicsLaws:
    @staticmethod
    def terzaghi_consolidation(settlement, time, cv=1.0, H=10.0):
        """Terzaghi一维固结理论: ∂u/∂t = cv * ∂²u/∂z²"""

    @staticmethod
    def settlement_rate_constraint(settlement, time, max_rate=0.01):
        """沉降速率约束: |∂u/∂t| < max_rate"""

    @staticmethod
    def monotonicity_constraint(settlement):
        """单调性约束: 沉降应该单调递增"""
```

**网络结构**:
- 输入维度: 4 (时间、温度、裂缝、振动)
- 隐藏层: [64, 128, 128, 64]
- 激活函数: Tanh
- 输出维度: 1 (沉降)

**损失函数**:
```
Total Loss = Data Loss + physics_weight * Physics Loss
Physics Loss = Terzaghi Loss + 0.5 * Rate Loss + 0.3 * Monotonicity Loss
```

**API接口**:
- `GET /api/ml/predict/pinn/<point_id>` - PINN预测
- `POST /api/ml/train/pinn/<point_id>` - PINN训练

### 2. Ensemble Learning 集成学习

**文件**: `backend/modules/ml_models/ensemble_predictor.py`

**核心方法**:
1. **Stacking** - 元学习器整合基学习器
2. **Weighted Average** - 加权平均 (权重 = 1 / MAE)
3. **Simple Average** - 简单平均

**基础模型**:
- ARIMA (传统时间序列)
- Informer (深度学习)
- STGCN (时空图卷积)
- PINN (物理约束)

**元学习器**:
- Ridge Regression (默认)
- Random Forest (可选)

**置信区间计算**:
```python
# 使用基础模型预测的标准差作为不确定性度量
std_preds = np.std(all_preds, axis=0)
lower = mean_preds - 1.96 * std_preds
upper = mean_preds + 1.96 * std_preds
```

**API接口**:
- `GET /api/ml/predict/ensemble/<point_id>` - 集成预测
  - 参数: `method` (stacking/weighted_average/simple_average)
  - 参数: `base_models` (逗号分隔的模型列表)

### 3. SHAP 可解释性分析

**文件**: `backend/modules/ml_models/explainability.py`

**核心功能**:
1. **SHAP值计算** - 量化每个特征对预测的贡献
2. **特征重要性排序** - 识别关键影响因素
3. **单个预测解释** - 解释具体预测结果
4. **统计摘要** - 特征的SHAP值统计

**支持的模型类型**:
- `tree` - 树模型 (RandomForest, XGBoost, LightGBM)
- `linear` - 线性模型 (LinearRegression, Ridge, Lasso)
- `deep` - 深度学习模型 (PyTorch, TensorFlow)
- `kernel` - 通用黑盒模型

**使用示例**:
```python
# 创建解释器
analyzer = ExplainabilityAnalyzer(model, model_type='tree')
analyzer.fit(X, feature_names)
analyzer.explain(X)

# 获取特征重要性
importance = analyzer.get_feature_importance()

# 解释单个预测
explanation = analyzer.explain_single_prediction(x, sample_idx=0)
```

**API接口**:
- `GET /api/ml/explain/<point_id>` - SHAP可解释性分析
  - 参数: `model_type` (tree/linear/deep/kernel)

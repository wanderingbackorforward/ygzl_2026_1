## 第一阶段：智能异常诊断 + 处置建议

### 实施时间
2026-03-05

### 功能实现

#### 1. 智能异常诊断
- 自动识别所有异常点（超阈值、变化率异常、趋势异常）
- 按严重程度分级（严重/高/中/低/正常）
- 显示异常原因、当前值、阈值、偏差
- 异常统计卡片展示

#### 2. 处置建议
- 根据异常自动生成处置建议
- 按优先级排序（紧急/高/中/低）
- 具体行动方案（巡检/监测/维修/上报）
- 建议统计卡片展示

2. `utils/analysis.ts` - 工具函数

4. `components/analysis/AnomalyStatCards.tsx` - 异常统计卡片
5. `components/analysis/AnomalyList.tsx` - 异常列表
6. `components/analysis/AnomalyDashboard.tsx` - 异常诊断主面板
7. `components/analysis/RecommendationCard.tsx` - 建议卡片
8. `components/analysis/RecommendationList.tsx` - 建议列表
9. `components/analysis/RecommendationDashboard.tsx` - 处置建议主面板
10. `components/analysis/index.ts` - 组件导出

#### 修改文件
- `pages/AdvancedAnalysis.tsx` - 添加2个新标签页

#### API调用
- `GET /api/analysis/v2/settlement` - 获取完整分析结果
- `GET /api/analysis/v2/settlement/anomalies` - 获取异常列表
- `GET /api/analysis/v2/settlement/recommendations` - 获取处置建议

### 构建结果
- 构建时间: 5.78秒
- 主包大小: 69.02 kB (gzip)

## 第二阶段：趋势预测与预警

### 实施时间
2026-03-05

### 功能实现

#### 1. 预测图表
- ECharts可视化预测曲线
- 历史数据（蓝色实线）+ 预测值（橙色虚线）
- 置信区间（橙色阴影区域）
- 模型信息展示（数据量、趋势强度、波动性、季节性、模型得分）
- 响应式设计，自动调整大小

#### 2. 高风险点位列表
- 自动计算风险等级
  - 严重: 预测值 ≤ 阈值
  - 高风险: 预测值 ≤ 80%阈值
  - 中风险: 预测值 ≤ 60%阈值
  - 低风险: 其他
- 显示预测最大沉降、阈值、偏差
- 计算到达阈值的天数
- 网格布局，颜色编码风险徽章
- 空状态处理（无高风险点位时显示安全提示）

#### 3. 模型对比表
- 对比多个模型性能（ARIMA, SARIMA, Prophet, LSTM, 线性回归）
- 显示评估指标：
  - MAE（平均绝对误差）
  - RMSE（均方根误差）
  - MAPE（平均绝对百分比误差）
  - AIC/BIC（信息准则）
- 高亮显示已选择的模型
- 指标说明图例
- 表格样式：斑马纹、悬停效果

#### 4. 预测Dashboard
- 统计卡片展示：
  - 预测点位数量
  - 高风险点位数量
  - 预测天数（30天）
  - 当前使用模型
- 点位选择器（按钮组）
- 模型对比按钮（可展开/收起）
- 集成预测图表、模型对比表、高风险点位列表
- 完整的加载、错误、空状态处理

### 技术实现

#### 新增组件（4个）
1. `components/analysis/PredictionChart.tsx` - 预测图表组件
2. `components/analysis/RiskPointList.tsx` - 高风险点位列表
3. `components/analysis/ModelComparisonTable.tsx` - 模型对比表
4. `components/analysis/PredictionDashboard.tsx` - 预测主面板

#### 修改文件
- `components/analysis/index.ts` - 添加新组件导出
- `pages/AdvancedAnalysis.tsx` - 添加"趋势预测"标签页

#### API调用
- `GET /api/ml/auto-predict/<point_id>?forecast_days=30` - 自动选择最优模型预测
- `GET /api/ml/compare-models/<point_id>?forecast_days=30` - 模型对比

#### 技术要点
- **ECharts配置**:
  - 深色主题，透明背景
  - 多系列数据（历史、预测、置信区间上下界）
  - 堆叠区域图实现置信区间
  - 自定义tooltip和图例
- **风险计算逻辑**:
  - 基于阈值百分比分级
  - 遍历预测值找到首次超限时间
  - 按风险等级和预测值排序
- **状态管理**:
  - 使用React Hooks（useState, useEffect）
  - 异步API调用与错误处理
  - 点位选择与模型对比切换

- **状态管理**: React Hooks
- **构建工具**: Vite 5.x

### 后端
- **框架**: Flask
- **ML库**: scikit-learn, statsmodels
- **数据处理**: pandas, numpy
- **数据库**: MySQL

### 部署
- **平台**: Vercel
- **环境**: Node.js + Python Serverless Functions

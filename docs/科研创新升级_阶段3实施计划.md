   - 破坏准则: τ = c + σ * tan(φ)

2. **Drucker-Prager模型**
   - 三维强度准则
   - 破坏准则: √J2 = α * I1 + k

3. **Terzaghi固结理论**
   - 一维固结
   - 沉降计算

4. **Biot固结理论**
   - 三维固结
   - 流固耦合

5. **Duncan-Chang模型**
   - 非线性弹性
   - 应力-应变关系

#### 简化模型

为了快速实现,我们使用简化的沉降预测模型:

```python
# 沉降 = f(荷载, 土体参数, 时间)
settlement = (load * H) / (E * A) * (1 - exp(-t/T))

其中:
- load: 荷载 (kN)
- H: 土层厚度 (m)
- E: 弹性模量 (MPa)
- A: 面积 (m²)
- t: 时间 (天)
- T: 固结时间常数 (天)
```

### 2.2 Kalman Filter数据同化

#### 原理

Kalman Filter用于融合物理模型预测和实际监测数据:

```
预测步骤:
x̂(k|k-1) = F * x̂(k-1|k-1) + B * u(k)
P(k|k-1) = F * P(k-1|k-1) * F^T + Q

更新步骤:
K(k) = P(k|k-1) * H^T * (H * P(k|k-1) * H^T + R)^-1
x̂(k|k) = x̂(k|k-1) + K(k) * (z(k) - H * x̂(k|k-1))
P(k|k) = (I - K(k) * H) * P(k|k-1)

其中:
- x̂: 状态估计(沉降)
- F: 状态转移矩阵
- B: 控制输入矩阵
- u: 控制输入(荷载)
- P: 误差协方差
- Q: 过程噪声协方差
- R: 测量噪声协方差
- K: Kalman增益
- z: 测量值(实际监测数据)
- H: 观测矩阵
```

#### 实现

```python
class KalmanFilter:
    def __init__(self, F, B, H, Q, R, x0, P0):
        self.F = F  # 状态转移矩阵
        self.B = B  # 控制输入矩阵
        self.H = H  # 观测矩阵
        self.Q = Q  # 过程噪声
        self.R = R  # 测量噪声
        self.x = x0  # 初始状态
        self.P = P0  # 初始协方差

    def predict(self, u):
        # 预测步骤
        self.x = self.F @ self.x + self.B @ u
        self.P = self.F @ self.P @ self.F.T + self.Q
        return self.x

    def update(self, z):
        # 更新步骤
        y = z - self.H @ self.x
        S = self.H @ self.P @ self.H.T + self.R
        K = self.P @ self.H.T @ np.linalg.inv(S)
        self.x = self.x + K @ y
        self.P = (np.eye(len(self.x)) - K @ self.H) @ self.P
        return self.x
```

### 2.3 情景模拟引擎

#### 支持的情景

1. **施工方案优化**
   - 问题: "先开挖A区还是B区?"
   - 模拟: 不同施工顺序下的沉降分布
   - 输出: 最优施工方案

2. **应急预案评估**
   - 问题: "如果发生暴雨,哪些点位最危险?"
   - 模拟: 极端降雨下的沉降变化
   - 输出: 高风险点位列表

3. **长期演化预测**
   - 问题: "5年后沉降分布?"
   - 模拟: 长期荷载作用下的沉降演化
   - 输出: 未来沉降分布图

#### 实现

```python
class ScenarioSimulator:
    def __init__(self, physics_model, kalman_filter):
        self.physics_model = physics_model
        self.kf = kalman_filter

    def simulate_construction_sequence(self, sequence):
        # 模拟不同施工顺序
        results = []
        for step in sequence:
            settlement = self.physics_model.predict(step)
            results.append(settlement)
        return results

    def simulate_extreme_weather(self, rainfall):
        # 模拟极端天气
        load_increase = self.calculate_load_from_rainfall(rainfall)
        settlement = self.physics_model.predict(load_increase)
        return settlement

    def simulate_long_term(self, years):
        # 模拟长期演化
        time_steps = years * 365
        settlements = []
        for t in range(time_steps):
            settlement = self.physics_model.predict(t)
            settlements.append(settlement)
        return settlements
```

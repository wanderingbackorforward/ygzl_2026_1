## Phase 1: 科学算法层（优先级最高）

**目标**：修复致命算法错误，建立GB规范合规基础

### 1.1 基础工具函数
- [x] 安装依赖库（fili, fft.js, mathjs）
- [x] 创建 `frontend/src/utils/vibration/` 目录结构
- [x] 创建 `types.ts` 定义所有类型接口

### 1.2 信号处理算法
- [x] 实现 Butterworth 带通滤波器（0.5-100Hz，4阶）
- [x] 实现 FFT 计算（复用或封装 fft.js）
- [x] 实现三轴合成PPV计算 `V(t) = √(Vx² + Vy² + Vz²)`
- [x] 实现振动持续时间计算（>0.1*PPV的时长）
- [x] 实现主频带宽度计算（FFT半功率带宽）

### 1.3 GB规范实现
- [x] 创建 GB 6722-2014 表2数据结构
- [x] 实现阈值查询函数（根据结构类型+频率）
- [x] 实现动态阈值修正（频率/距离/结构状态）
- [x] 实现 Sadovsky 公式（预测远场振动）
- [x] 实现安全距离计算器

### 1.4 评分引擎
- [x] 实现安全评分算法（基于GB规范 + Miner累积损伤）
- [x] 实现预警等级判定（safe/caution/danger）
- [x] 实现施工建议生成（规则引擎）

### 1.5 单元测试
- [ ] 滤波器测试（对比标准信号）
- [ ] PPV计算测试（对比专业软件）
- [ ] 阈值计算测试（覆盖7种结构类型）
- [ ] 评分算法测试（对比专家评估）

**验收标准**：
- PPV计算结果与专业振动分析软件（如DASP）误差<5%
- 阈值计算符合GB 6722-2014表2
- 安全评分与人工专家评估一致性>90%

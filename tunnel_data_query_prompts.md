## 数据验证和质量要求提示词

```
对提供的隧道工程数据，请进行以下验证：

1. **数据真实性验证**
   - 核实数据来源的权威性
   - 确认参数的合理性范围
   - 检查数据的逻辑一致性

2. **技术参数验证**
   - 推进力：通常在10000-20000 kN范围
   - 刀盘扭矩：根据直径不同，通常在2000-6000 kN·m
   - 推进速度：城市隧道通常在20-80 mm/min
   - 各项压力参数应该在合理范围内

3. **坐标验证**
   - 经纬度坐标应该符合实际地理位置
   - 工程坐标系转换正确
   - 隧道轨迹连续性检查

4. **数据完整性**
   - 时间序列的连续性
   - 23维参数的完整性
   - 缺失值和异常值处理

请提供验证报告和数据质量评估。
```

## 使用说明

1. **选择合适的提示词**：根据需要查询的具体隧道和参数类型选择对应的提示词

2. **调整查询重点**：可以根据实际需求调整提示词中的重点内容

3. **验证数据质量**：获取数据后务必进行真实性和合理性验证

4. **格式化输出**：要求AI以结构化格式输出数据，便于后续处理

5. **多源验证**：建议通过多个渠道验证关键参数的准确性

## 注意事项

- 优先选择官方发布的技术资料和学术论文
- 注意数据的时间戳和适用条件
- 考虑不同工程之间的技术差异
- 关注数据的版权和使用限制
- 确保数据的可追溯性和验证性
